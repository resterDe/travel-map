# Travel-Map 功能需求文档

## 一、产品概述

**产品名称**: Travel-Map 旅行地图相册  
**产品类型**: 桌面端本地应用  
**核心价值**: 通过本地文件夹管理旅行照片，自动按地级市分类，在地图上可视化展示，构建个人旅行记忆

---

## 二、用户群体

| 用户类型 | 需求描述 |
|----------|----------|
| 摄影爱好者 | 管理大量旅行照片，按地区分类查看 |
| 旅行者 | 记录足迹，展示旅行地图合集 |
| 家庭用户 | 整理家庭旅行照片，回顾美好时光 |
| 驴友 | 标记我去过的城市，展示旅行轨迹 |

## 四、用户需求详细描述

### 4.1 根目录设置与管理

#### 4.1.1 默认目录

- 用户可指定任意盘符/路径作为资源根目录（如 `D:\travel-map-resource`）
- 首次安装/启动时，系统自动初始化默认目录结构
- 防止目录不存在，确保后续功能正常运行

#### 4.1.2 目录迁移

- 用户可在设置页面修改根目录
- 迁移时保留原有数据（复制/移动模式可选）
- 迁移完成后自动更新数据库配置

### 4.2 照片存储结构

用户按以下目录结构组织照片：

```
资源根目录 (如 D:\travel-map-resource)/
├── 湛江市/
│   ├── 雷州一日游_2024-03-15_探索雷州古城/
│   │   ├── IMG_001.jpg
│   │   ├── IMG_002.png
│   │   └── VIDEO_001.mp4
│   └── 东海岛夏日_2024-07-20_海滩度假/
│       └── ...
├── 广州市/
│   └── ...
└── 北京市/
    └── ...
```

**目录命名规则**: `{用户备注}_{日期}_{描述}`

### 4.3 城市目录创建流程

| 步骤 | 用户操作 | 系统响应 |
|------|----------|----------|
| 1 | 点击地图上的地级市（如"湛江市"） | 检查该城市目录是否存在 |
| 2 | 目录不存在时，弹出创建对话框 | 显示创建表单 |
| 3 | 填写目录备注（如"雷州一日游"） | 实时预览目录名 |
| 4 | 选择旅行日期 | 日期选择器 |
| 5 | 填写描述（可选） | 描述输入框 |
| 6 | 点击确认 | 创建目录并跳转到该目录 |

**目录命名示例**:
- 用户备注: 雷州一日游
- 选择时间: 2024-03-15
- 描述: 探索雷州古城
- 最终目录名: `雷州一日游_2024-03-15_探索雷州古城`

### 4.4 目录管理功能

| 功能 | 描述 |
|------|------|
| 目录重命名 | 在相册页面可重命名目录 |
| 目录删除 | 删除目录及内容（需二次确认） |
| 目录备注编辑 | 修改目录的备注、时间、描述 |
| 查看目录信息 | 显示创建时间、文件数量、总大小 |

### 4.2 地图交互需求

| 场景 | 用户操作 | 系统响应 |
|------|----------|----------|
| 查看地图 | 打开应用 | 显示中国地图，标注有照片的城市 |
| 选择城市 | 点击地级市 | 跳转到该城市相册页面 |
| 查看照片 | 点击照片缩略图 | 弹出大图查看器 |
| 返回地图 | 点击返回/关闭 | 返回地图主界面 |

### 4.3 视频轮播需求

| 场景 | 用户操作 | 系统响应 |
|------|----------|----------|
| 播放视频 | 点击视频项 | 开始播放视频 |
| 自动轮播 | 开启自动播放 | 每N秒切换到下一个视频 |
| 手动切换 | 左右滑动 | 切换到上一个/下一个视频 |
| 全屏观看 | 点击全屏按钮 | 进入全屏播放模式 |
| 退出全屏 | 按ESC或点击退出 | 退出全屏模式 |

### 4.4 BGM需求

| 场景 | 用户操作 | 系统响应 |
|------|----------|----------|
| 添加音乐 | 点击添加按钮 | 打开文件选择器 |
| 播放音乐 | 点击播放按钮 | 开始播放当前曲目 |
| 切换音乐 | 点击上一曲/下一曲 | 切换播放曲目 |
| 调节音量 | 拖动音量滑块 | 调整音量大小 |

## 五、非功能性需求

### 5.1 性能要求

| 指标 | 要求 |
|------|------|
| 应用启动时间 | < 3秒 |
| 地图加载时间 | < 2秒 |
| 照片缩略图加载 | < 100ms/张 |
| 视频切换响应 | < 500ms |
| 内存占用 | < 500MB（正常使用） |

### 5.2 兼容性要求

| 平台 | 最低版本 |
|------|----------|
| Windows | Windows 10 |
| macOS | macOS 10.15 |
| Linux | Ubuntu 20.04 |

### 5.3 数据安全

- 所有数据存储在本地，不上传云端
- 支持数据库备份/导出
- 缩略图缓存可清理

## 七、边缘情况处理

| 场景 | 处理方式 |
|------|----------|
| 空目录（无照片） | 显示空状态提示 |
| 目录不存在 | 提示用户设置正确目录 |
| 损坏的图片文件 | 跳过并记录日志 |
| 超大视频文件 | 流式加载，不全部加载到内存 |
| 大量照片（>10000） | 分页加载，虚拟滚动 |
| 无网络（地图瓦片） | 使用离线地图数据 |
