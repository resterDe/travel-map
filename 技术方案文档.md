# Travel-Map 技术方案文档

## 一、项目概述

**项目名称**: Travel-Map  
**项目类型**: 桌面端本地照片管理应用  
**核心目标**: 管理本地旅行照片，构建个人旅行地图集，实现照片分类、地级市展示、视频轮播、BGM背景音乐等功能  
**部署方式**: 本地部署（零云资源占用）/ 可扩展云服务部署

### 3.1 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                    Electron Main Process                │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────┐  │
│  │  FileSystem │  │   SQLite    │  │  Window Manager │  │
│  │   Manager   │  │   Database  │  │                 │  │
│  └─────────────┘  └─────────────┘  └─────────────────┘  │
│         ↑               ↑                               │
│         └───────────────┼───────────────────────────────┤
│                         │ IPC Communication             │
├─────────────────────────┼───────────────────────────────┤
│                    Renderer Process                      │
│  ┌─────────────────────────────────────────────────┐    │
│  │                   Vue 3 App                     │    │
│  │  ┌─────────┐  ┌─────────┐  ┌─────────────────┐  │    │
│  │  │ Router  │  │  Pinia  │  │  Components     │  │    │
│  │  │         │  │ Store   │  │ (Map/Gallery)   │  │    │
│  │  └─────────┘  └─────────┘  └─────────────────┘  │    │
│  └─────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────┘
```

### 3.2 模块划分

#### 主进程模块 (Main Process)

| 模块 | 职责 |
|------|------|
| `WindowManager` | 窗口创建、管理、多窗口支持 |
| `FileSystemService` | 扫描本地目录、文件监控、图片/视频读取 |
| `DatabaseService` | SQLite数据库CRUD操作 |
| `ThumbnailService` | 图片缩略图、视频封面生成 |
| `MediaScanner` | 媒体文件扫描、元数据提取 |

#### 渲染进程模块 (Renderer)

| 模块 | 职责 |
|------|------|
| `MapView` | 中国地图渲染、地级市交互 |
| `GalleryView` | 照片/视频网格展示 |
| `CarouselView` | 轮播组件（视频/图片） |
| `AudioPlayer` | BGM播放控制 |
| `SettingsView` | 配置管理界面 |

### 3.3 数据流向

```
本地目录 ──扫描──> SQLite数据库 ──查询──> Vue组件
     │                                      │
     └─ 缩略图缓存 ───────────────────────────┘
```

## 五、核心功能实现方案

### 5.1 目录管理

- 用户自定义设置资源根目录（如 `D:\travel-map-resource`）
- 首次启动时自动初始化根目录
- 支持目录迁移（复制/移动模式）
- 目录命名规则: `{用户备注}_{日期}_{描述}`
- 示例: `雷州一日游_2024-03-15_探索雷州古城`

### 5.2 地图渲染

- 使用 **Leaflet** 加载中国地图（GeoJSON）
- 地级市区域绑定点击事件
- 根据照片数量实现区域颜色热力效果
- 点击区域跳转到相册页面

### 5.2 照片管理

- 扫描指定本地目录（按地级市分类）
- 使用 `Sharp` 生成缩略图缓存
- 支持图片格式：JPG, PNG, WEBP, HEIC
- 支持视频格式：MP4, MOV, AVI

### 5.3 视频轮播

- 使用 `Swiper` 实现触摸/鼠标滑动
- 自动播放/暂停控制
- 全屏模式支持
- 视频封面预览

### 5.4 BGM功能

- 使用 `Howler.js` 管理音频播放
- 支持MP3/OGG音频格式
- 背景持续播放
- 音量控制、切换曲目

## 六、数据库设计

### 6.1 核心表结构

```sql
-- 地级市表
CREATE TABLE cities (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    code VARCHAR(10) UNIQUE NOT NULL,    -- 行政区划代码
    name VARCHAR(50) NOT NULL,           -- 城市名称
    province VARCHAR(50) NOT NULL,      -- 所属省份
    latitude REAL,                      -- 中心纬度
    longitude REAL,                      -- 中心经度
    photo_count INTEGER DEFAULT 0,      -- 照片数量
    video_count INTEGER DEFAULT 0,       -- 视频数量
    has_directory INTEGER DEFAULT 0,    -- 是否有目录
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 旅行目录表 (城市子目录)
CREATE TABLE trips (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    city_id INTEGER NOT NULL,            -- 关联城市
    dirname VARCHAR(100) NOT NULL,       -- 目录名 (用户备注)
    trip_date DATE,                      -- 旅行日期
    description VARCHAR(200),             -- 旅行描述
    folder_name VARCHAR(150) NOT NULL,   -- 最终目录名 (系统生成)
    full_path VARCHAR(500) NOT NULL,     -- 完整路径
    photo_count INTEGER DEFAULT 0,        -- 照片数量
    video_count INTEGER DEFAULT 0,       -- 视频数量
    total_size INTEGER DEFAULT 0,        -- 总大小(字节)
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (city_id) REFERENCES cities(id)
);

-- 媒体文件表
CREATE TABLE media (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    trip_id INTEGER NOT NULL,            -- 关联旅行目录
    city_id INTEGER NOT NULL,            -- 关联城市
    filename VARCHAR(255) NOT NULL,      -- 文件名
    filepath VARCHAR(500) NOT NULL,      -- 文件路径
    type VARCHAR(10) NOT NULL,           -- 类型: image/video
    format VARCHAR(10),                  -- 格式: jpg/mp4等
    width INTEGER,                       -- 宽度
    height INTEGER,                      -- 高度
    size INTEGER,                        -- 文件大小
    duration REAL,                       -- 视频时长(秒)
    thumbnail_path VARCHAR(500),        -- 缩略图路径
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (trip_id) REFERENCES trips(id),
    FOREIGN KEY (city_id) REFERENCES cities(id)
);

-- BGM音乐表
CREATE TABLE bgm (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    filename VARCHAR(255) NOT NULL,
    filepath VARCHAR(500) NOT NULL,
    title VARCHAR(100),
    artist VARCHAR(100),
    duration REAL,
    is_default BOOLEAN DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 应用设置表
CREATE TABLE settings (
    key VARCHAR(50) PRIMARY KEY,
    value TEXT
);
```
