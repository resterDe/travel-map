# Travel-Map 设计思路文档

## 一、设计理念

### 1.1 核心设计目标

- **沉浸式体验**: 让用户专注于旅行回忆，而非工具本身
- **现代化视觉**: 符合当代审美，简洁大方
- **旅行主题**: 传递探索世界的美好情感

### 1.2 设计关键词

| 关键词 | 含义 |
|--------|------|
| **探索** | 地图承载对未知的好奇 |
| **回忆** | 照片承载旅途的美好 |
| **自然** | 色彩灵感源于自然风光 |
| **简洁** | 去除干扰，聚焦核心内容 |

## 二、视觉设计

### 2.1 色彩系统

#### 主色调 (Primary Colors)

| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| 深海蓝 | `#1E3A5F` | 主色，品牌元素 |
| 天空蓝 | `#4A90D9` | 交互元素，高亮 |
| 云雾白 | `#F5F7FA` | 浅色背景 |
| 雾霾灰 | `#6B7B8C` | 次要文字 |

#### 辅助色 (Accent Colors)

| 颜色名称 | 色值 | 用途 |
|----------|------|------|
| 晨曦金 | `#F5A623` | 徽章、提示 |
| 草原绿 | `#4CAF50` | 成功状态 |
| 夕阳红 | `#E74C3C` | 警告/删除 |
| 薰衣草紫 | `#9B59B6` | BGM音乐相关 |

#### 深色主题

| 颜色名称 | 色值 | 用途 |
|----------|------|------|

| 星光白 | `#E0E6ED` | 深色文字 |
| 极光蓝 | `#00D4FF` | 深色高亮 |

```

## 三、UI组件设计

### 3.1 顶部导航栏

```
┌─────────────────────────────────────────────────────────────┐
│ [🗺️ Travel-Map]          [地图] [相册] [🎵 BGM]    [⚙️ 设置] │
└─────────────────────────────────────────────────────────────┘
```

**设计规范**:
- 高度: 56px
- 背景: 白色 / 深色模式深夜蓝
- 阴影: 0 2px 8px rgba(0,0,0,0.08)
- Logo: 旅行图标 + 应用名称
- 导航项: 选中状态下划线高亮

### 3.2 地图页面

**布局**:
```
┌─────────────────────────────────────────┐
│  顶部导航栏                              │
├─────────────────────────────────────────┤
│                                         │
│                                         │
│            中国地图区域                 │
│         (全屏铺满，边缘留白16px)         │
│                                         │
│                                         │
├─────────────────────────────────────────┤
│  [城市名称] [照片数] [最后更新]  [筛选▼] │
└─────────────────────────────────────────┘
```

**地图样式**:
- 海洋: `#E8F4F8`
- 陆地: `#FFFFFF`
- 边界线: `#D0D7DE`
- 悬浮效果: 亮度+10%
- 选中效果: 天空蓝边框 + 阴影

- 悬浮效果: 放大1.02 + 阴影
- 视频标识: 左上角播放图标

### 3.4 轮播组件

**布局**:
```
┌─────────────────────────────────────────┐

- 自动播放间隔: 5秒 (可配置)
- 过渡动画: 300ms 滑动

## 四、交互设计

### 4.1 页面过渡

| 场景 | 过渡效果 |
|------|----------|
| 页面切换 | 淡入淡出 200ms |
| 地图→相册 | 地图缩小+淡出 300ms |
| 相册→轮播 | 滑入效果 250ms |
| 弹窗打开 | 缩放+淡入 200ms |

### 4.2 交互动效

| 元素 | 效果 | 持续时间 |
|------|------|----------|
| 按钮悬浮 | 轻微上移 + 阴影增强 | 150ms |
| 卡片悬浮 | 轻微放大 (1.02) | 200ms |
| 缩略图点击 | 轻微缩小 (0.98) | 100ms |
| 地图区域悬浮 | 填充色变化 + 提示 | 150ms |
| 加载过程 | 骨架屏动画 | - |

### 4.3 手势支持

| 场景 | 手势 |
|------|------|
| 图片浏览 | 左右滑动切换 |
| 视频轮播 | 左右滑动切换 |
| 地图操作 | 单指拖拽/缩放 |
| 图片预览 | 双指缩放/拖拽 |

## 十、实施建议

### 10.1 设计系统构建顺序

1. 色彩系统 → 基础CSS变量
2. 字体系统 → 全局字体设置
3. 组件库 → 基础组件 (按钮/输入框/卡片)
4. 布局组件 → 导航/网格/列表
5. 页面模板 → 地图/相册/设置

### 10.2 组件开发优先级

| 优先级 | 组件 |
|--------|------|
| P0 | 按钮、输入框、卡片、导航栏 |
| P0 | 地图组件、照片网格 |
| P1 | 图片预览、轮播组件 |
| P1 | BGM播放器 |
| P2 | 骨架屏、空状态、加载动画 |
